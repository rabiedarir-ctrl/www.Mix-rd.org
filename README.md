# Mix Platform — ناطق www.mix-rd.org

مرحبًا بك في **Mix Platform**، المنصة الرقمية الموحدة لإدارة المحتوى، المحفظة، الألعاب، والميتافيرس.  

هذا المستودع يحتوي على **نسخة ناطق رسمي للمشروع** جاهزة للعمل مباشرة على GitHub.

---

## الرابط الرسمي للناطق

**[www.mix-rd.com](https://www.mix-rd.org)**

> هذا العنوان يمثل واجهة المشروع الأساسية ويعمل كنظام ناطق على GitHub Pages.

---

## الميزات الرئيسية

- واجهة تشغيل رقمية موحدة
- إدارة المحفظة الرقمية
- عرض المحتوى الاجتماعي
- متجر عناصر داخل المنصة
- نظام الألعاب والميتافيرس
- دعم تحديثات مباشرة عبر GitHub

## كيفية التشغيل

1. انسخ المستودع أو اضغط **Fork**.
2. تأكد من وضع **CNAME** مع اسم الناطق www.mix-rd.org.
3. ارفع الملفات على **GitHub Pages**.
4. افتح الرابط:أو مباشرة عبر
    **www.mix-rd.org**
5. بعد ضبط DNS.

## تطوير المنصة

- يمكنك تعديل **index.html** لتغيير محتوى الناطق.  
- استخدم **static/main.js** لإضافة أي وظائف ديناميكية.  
- استخدم **static/style.css** لتغيير التصميم.
